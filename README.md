# 📦 Inventory Management System

The **Inventory Management System** is a web-based application designed to manage and track inventory efficiently. It allows users to add, update, delete, and view product records while monitoring stock availability.  
This project is ideal for **college projects**, **small businesses**, and **learning PHP-based web development**.

---

## 🚀 Features

- 🔐 User Authentication
- 📋 Add, Update, Delete Products
- 📦 Inventory Stock Management
- 🔎 Product Search
- 🧾 Category-wise Product Organization
- 📊 Stock Status Monitoring
- 📱 Responsive UI using Bootstrap

---

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- Bootstrap

### Backend
- PHP

### Database
- MySQL (phpMyAdmin)
