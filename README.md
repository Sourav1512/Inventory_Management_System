It is suitable for **academic projects**, **small businesses**, and **learning full-stack web development with PHP**.

## 📖 Overview

The **Inventory Management System (IMS)** provides a centralized platform to manage inventory records.  
Users can maintain product information, monitor stock availability, and perform basic inventory operations through a clean and responsive user interface.

---

## 🚀 Features

- Secure Authentication System  
- Product Management (Add / Edit / Delete)  
- Inventory Stock Tracking  
- Category-wise Product Management  
- Search & Filter Products  
- Stock Status Monitoring  
- Responsive UI with Bootstrap
